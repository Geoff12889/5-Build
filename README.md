# 5-Build

## Module 5 Build
The Indie Film Collective is a multi-page webpage meant to tell the community about its services. The HTML is implemented with a mobile-first design that is both semantic and validated (passes W3C validation). The CSS is responsive, expanding to a friendly desktop interface. No JavaScript is used.
